Procedural Image Nodal-Generator. 100% free &amp; open source.


## Try it now
[PING](https://ping.bubblebirdstudio.com/)


## Features
- Easily generates seamless textures
- Support for 2D, 3D and 2D-animated textures
- Loopable 2D-animated textures (with realtime preview)
- PNG export
- Realtime 3D viewer
- Quick start with cool templates
- 30+ nodes, endless possibilites
- Fast (uses compute shaders with WebGPU)

## Limitations
- Ping requires a modern browser that fully supports WebGPU in order to work. Check this [compatibility table](https://developer.mozilla.org/en-US/docs/Web/API/WebGPU_API#browser_compatibility).
- Texture size is limited by the implementation of WebGPU. Tested up to 16384 x 4096 pixels and 8192 x 8192 pixels.
- Currently only exports 8bit per channel PNG files
